## frontend 
```bash
cd frontend
```

## Installation

Use the package manager [pnpm]([https://pip.pypa.io/en/stable/](https://pnpm.io/installation)) to install packages is a lot faster then npm.

```bash
pnpm install
```

or use normal npm but is slow (not recommended)
```bash
npm install
```
## Run Command

```bash
pnpm dev
```
or
```bash 
npm run dev
```
